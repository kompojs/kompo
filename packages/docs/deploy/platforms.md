# Platforms de Déploiement

L'architecture Kompo est optimisée pour le déploiement serverless et edge.

## Stack Recommandée

- **Frontend** : [Vercel](https://vercel.com) pour une intégration Next.js parfaite.
- **Base de Données** : [Neon](https://neon.tech) pour un PostgreSQL serverless avec branching.
- **Backend/Workers** : [Railway](https://railway.app) pour les services Docker et Redis.

## Intégration

Chaque adapter Kompo est conçu pour être facilement configurable sur ces plateformes via les variables d'environnement standards.
