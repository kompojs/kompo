# Architecture Graph

L'Architecture Graph du Kompo Studio vous donne une vue d'ensemble en temps réel de votre projet.

## Fonctionnalités

- **Navigation interactive** : Cliquez sur un domaine pour voir ses ports et entités.
- **Détection des cycles** : Soyez alerté immédiatement si une règle d'architecture est violée.
- **Export** : Exportez votre graphe pour vos présentations techniques.

## Accès

Lancez le studio localement pour voir le graphe de votre projet actuel.
