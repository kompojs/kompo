# Dependency Viewer

Le Dependency Viewer vous aide à maintenir un monorepo sain.

## Fonctionnalités

- **Package Graph** : Voyez quel package dépend de quel autre.
- **Version Mismatch** : Identifiez les versions divergentes d'une même bibliothèque entre vos apps.
- **Bundle Analysis** : Estimez l'impact de chaque package sur votre bundle final.
