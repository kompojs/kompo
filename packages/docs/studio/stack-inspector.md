# Stack Inspector

Vérifiez l'état de santé de votre stack technologique en un clin d'œil.

## Fonctionnalités

- **Live Status** : Vérifiez la connexion à votre base de données ou RPC.
- **Config Viewer** : Visualisez comment chaque port est branché à son adapter.
- **Doctor Integration** : Lancez des réparations directement depuis l'interface.
