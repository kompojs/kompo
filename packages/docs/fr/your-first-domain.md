# Votre Premier Domaine

Créez votre premier domaine avec la CLI :

```bash
kompo add domain nft-collection
```

La CLI vous guidera pour définir le use case, les ports et l'entité. Consultez le [guide en anglais](/your-first-domain) pour plus de détails.
